# Fish Tank Task Tracker

A cozy React + Vite app where every task becomes a fish in a fish tank.

## Features

- Add tasks as fish
- Click fish to mark tasks complete
- Delete tasks
- Filter by all / active / completed
- Local storage persistence
- Animated aquarium UI

## Tech stack

- React
- Vite
- Framer Motion
- Lucide React
- Plain CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
